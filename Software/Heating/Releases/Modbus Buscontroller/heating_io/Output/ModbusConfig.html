Module Parameter Table

The module parameter table lists the parameters of the buscontroller and for each configured module, such as the function model in which the module should be operated and the product code of the module expected for a specific slot. These parameters can be read through the functions 3, 4, 23 and written by the functions 6, 16 and 23. The parameter ProductCode represents the value which uniquely identifies the module, which is also known as Hardware ID or B&R ID code.

  • Slot shows the slot where the module is plugged (Slot 0 is preserved for the controller).
  • Module shows the hardware name of the module.
  • Parameter displays the parameter name.
  • Address displays the address of the parameter.
  • Datatype displays the datatype of the parameter.
  • Size in bytes of the parameter according to the datatype.
  • Value shows the value which is configured for the parameter.

Register Configuration Table

The register configuration table shows the configuration values for the I/O module registers. These records can be read through the functions 3, 4, 23 and written by the functions 6, 16 and 23.

  • Length shows the number of entries in a configuration block.
  • Module contains the slot numbers of the modules by which a configuration block is referenced.
  • Address displays the address of a register configuration record.
  • Type shows the type of the register to configure.
  • Size displays the size in bytes of the register to configure.
  • Value contains the initialization value of the register.
  • Record shows the configuration entry in hex as it is written to the controller.
