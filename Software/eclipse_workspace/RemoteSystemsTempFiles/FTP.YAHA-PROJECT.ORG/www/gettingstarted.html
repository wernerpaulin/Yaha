Getting started

1. Get the hardware

The currently supported hardware platform is Raspberry Pi. Although it runs also on generation 2, generation 3 is preferred because of performance and built-in wifi. The hardware can be bought at various places including amazon.com or other online shops like reichelt.de, conrad.at or elv.de
Minimum equipment:
  • Pi board
  • Micro SD card 8 GB or more
  • Housing
  • Power supply 2A
  • Heat sink set
Although the Pi is often used headless (no monitor), a monitor, keyboard and mouse is highly recommended during configuration.
If Enocean communication should be used an additional add-on board for the GPIO connector is required and can also be bought through the online shops mentioned above.

2. Install Raspbian OS

The Raspberry Pi operating system, a Linux derivate of Debian is available on the Raspberry Pi website.
The website also explains how to install it on a SD card.

3. Configuration of Raspbian OS

Although Raspbian starts out of the box, it does need some configuration to be suitable for Yaha. All necessary steps are documented in a Raspberry Pi setup guideline as mind map. It guides you step by step through this process and is available in the download center as PDF or Xmind. Xmind can be downloaded free of charge from: xmind.net.
Most of the steps are actually useful for any purpose of use.

4. Copy Yaha to Raspberry Pi

The following zip files need to be expanded and its content to be copied onto the Pi. The best way of doing it is using filezilla which can be downloaded free of charge from filezilla-project.org.
Yaha allows you to split modules up on several PIs. For stability purpose the heating modules runs on a separate Pi. However all PIs have the destination path in common:
Package Destination path on Pi
Python /home/pi/Yaha
CGI /usr/lib/cgi-bin
www /var/www
All files are available in the download center.

4. Customize your Yaha

You most likely want or need to tailor Yaha to your requirements.
There are two options:
  1. You are happy with the functionality of existing modules
    Then all you need is to define your I/Os by editing the Excel file "Yaha PDI XML generator.xlsm" and create all I/O xml-files (launch macro inside Excel). Then browse the xml configurations of all modules you want to use and edit them according to your requirements. Download all again to the Pi and reboot it.
  2. You want to change modules behaviour and want to create new modules
    For this you need to install Eclipse and PyDev to program in Python.
