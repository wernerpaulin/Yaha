Installation

You can install Showkase to the root of your website (e.g. http://www.example.com/) or to a specified subdirectory of your website (e.g. http://www.example.com/mysite/).

To install to the the root of your website:

  1. Upload the contents of the ‘showkase’ folder to your server web root folder.
  2. Browse to http://www.example.com/admin/ and click the 'Setup' button to run the installer.
  3. Showkase is now ready to use! Click 'Add New Page' to get started.
  4. Note that your temporary login is username: 'admin', password: 'admin'.

To install to a specified subdirectory of your website:

  1. Create a new folder on your web server. The folder name will be used in your web page addresses so keep it short and use only letters and numbers (no spaces), for example ‘mysite’.
  2. Upload the contents of the ‘showkase’ folder to your new folder.
  3. If you wish to use the Showkase favicon and apple touch icon then copy favicon.ico and apple-touch-icon.png to the web root.
  4. Browse to http://www.example.com/mysite/admin/ and click the 'Setup' button to run the installer.
  5. If you get error messages then change the permissions on your new folder to 0777, reload the browser page and click 'Setup' again.
  6. Showkase is now ready to use! Click 'Add New Page' to get started.
  7. Note that your temporary login is username: 'admin', password: 'admin'.

Upgrading Showkase

To upgrade an existing Showkase site to a newer version of Showkase:

  1. Backup your existing web site. If you have made any changes to the standard Showkase themes then make sure you include the ‘_themes’ directory in your backup.
  2. Delete the old ‘admin’ directory inside your installation directory and replace with the new one.
  3. Delete the old ‘_themes’ directory and replace with the new one.
  4. Delete the old ‘index.php’ and replace with the new one.
  5. Don‘t delete anything else.
  6. If necesssary, re-instate your changes to the Showkase themes.
  7. Republish
